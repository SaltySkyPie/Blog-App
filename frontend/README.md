## Description

React + TypeScript + Vite

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# development with gql generator
$ npm run dev-gql

# production mode
$ npm run build

# serve production build
$ npm run preview

# generate types from gql schema
$ npm run generate-gql
```
